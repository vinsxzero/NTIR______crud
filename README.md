# Formulário de Inefetividade Terapeutica NABOTA
Este projeto foi desenvolvido para aumentar a coleta de dados sobre inefetividade terapeutica do produto.
O aplicativo consiste em nas operações de criação e visualização somente pela parte administradora.

tutorial 
1. Configure o banco de dados
   back > ormconfig.json
2. Instale os módulos
   cd back > yarn
3. execute o programa
   yarn dev
